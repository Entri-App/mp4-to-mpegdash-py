# mp4-to-mpegdash-py
Python Script to convert a MP4 file into onDemand MPEG-DASH
## Dependencies 
* ffmpeg - https://www.ffmpeg.org/ 
* MP4Box - https://gpac.wp.mines-telecom.fr/mp4box/

## Usaage 
`$ python transcode.py $FILENAME`
